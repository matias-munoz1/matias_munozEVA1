Set de Bloques de Construcción

Características del Set de Bloques de Construcción:

  • Incluye 100 bloques de colores de diferentes formas y tamaños.
  • Fomenta el desarrollo de habilidades motoras y creatividad.
  • Material seguro y no tóxico.
  • Adecuado para niños mayores de 3 años.

Este set de bloques de construcción es ideal para que los niños desarrollen su imaginación y habilidades de resolución de problemas. Los bloques vienen en una variedad de colores y formas, lo que permite construir todo lo que la imaginación permita.
